# Paradise Nursery 🌿

## Project Overview

Paradise Nursery is a React-based web application that allows users to browse different houseplants and add them to a shopping cart. The application demonstrates the use of React components, React Router for navigation, and Redux Toolkit for state management.

## Features

* Landing page with background image and company introduction.
* Product listing page displaying different houseplants with images, names, and prices.
* Plants are organized into categories.
* Add to Cart functionality for each plant.
* Cart icon that dynamically updates with the number of items added.
* Shopping cart page showing selected plants with quantity controls.
* Ability to increase, decrease, or remove items from the cart.
* Total cart amount calculation.
* Checkout button with a “Coming Soon” message.

## Technologies Used

* React.js
* Redux Toolkit
* React Router DOM
* JavaScript
* HTML & CSS

## How to Run the Project

1. Clone the repository.
2. Install dependencies using:

npm install

3. Start the development server:

npm start

The application will run at http://localhost:3000
