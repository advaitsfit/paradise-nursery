# e-plantShopping – Paradise Nursery 🌿

## Overview

Paradise Nursery is a React application that allows users to browse houseplants and add them to a shopping cart. The app demonstrates React components, React Router for navigation, and Redux Toolkit for state management.

## Features

* Landing page with background image and company introduction.
* Product listing page grouped by plant categories.
* Add to Cart functionality with disabled button after adding.
* Cart icon showing total number of items.
* Shopping cart page with quantity controls and delete option.
* Total cart amount calculation.
* Checkout button displaying “Coming Soon”.
* Continue shopping navigation back to plant list.

## Technologies Used

* React.js
* Redux Toolkit
* React Router DOM
* JavaScript
* HTML / CSS
